Weeded Library Data Knowledge Graph Explorer

This interactive visualization represents withdrawn print book data from Hunter College Libraries as a knowledge graph. Search across bibliographic metadata and explore connections among records. Results are retrieved from RDF stored in TriplyDB via a SPARQL endpoint and rendered as an interactive network.
